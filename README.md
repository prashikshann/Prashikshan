# PRASHISKSHAN - Academia-Industry Interface Platform

AI-powered platform connecting students with internships and industry opportunities.

## Project Structure

```
Prashikshan_02/
├── frontend/          # React + TypeScript + Vite (PWA-ready)
│   ├── src/           # Source code
│   │   ├── components/    # UI components
│   │   ├── pages/         # Route pages
│   │   ├── hooks/         # Custom hooks
│   │   ├── integrations/  # Supabase integration
│   │   └── lib/           # Utilities
│   ├── public/        # Static assets & PWA files
│   └── package.json   # Frontend dependencies
├── backend/           # Python Flask API
│   ├── app.py
│   └── requirements.txt
└── supabase/          # Supabase configuration
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev      # Development server
npm run build    # Production build
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
